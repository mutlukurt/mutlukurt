## 🧠 Why Claude Code (CLI)?  

I’ve explored Cursor, Bolt, Lovable, Emergent, and Rork — but I chose **Claude Code CLI** as my main environment because it is:  
- **Lightweight & distraction-free** → CLI-first workflow, no overhead.  
- **Reliable & error-resistant** → generates cleaner code for complex UIs.  
- **Fast & iterative** → scaffold, refactor, and deploy in hours, not weeks.  

This choice reflects my philosophy: **simplicity, speed, and reliability**.
